
# Executing Nodes
Double clicking on a node, it is executed andGetting results 

### Console Output
<kbd>CTRL + Enter</kbd>
### Result Explorer
<kbd>CTRL + E</kbd>
### Evaluating a Node
<kbd>CTRL + R</kbd> to get its result
